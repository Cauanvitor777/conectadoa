# ConectaDoa

## Projeto Integrador Extensionista I (PIE I)

O ConectaDoa é um sistema desenvolvido como projeto acadêmico para auxiliar uma ONG fictícia no gerenciamento de doadores, doações e itens recebidos.

## Objetivo

O sistema tem como objetivo facilitar o cadastro de doadores, o registro das doações, o controle dos itens doados e a geração de relatórios para apoiar a administração da ONG.
